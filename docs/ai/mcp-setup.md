# Настройка MCP (Model Context Protocol) серверов для Kopim

Использование MCP-серверов позволяет ИИ-агентам выполнять глубокий анализ кода, использовать инструменты рефакторинга и получать доступ к актуальной документации непосредственно во время работы.

> [!NOTE]
> MCP-серверы являются исключительно вспомогательными инструментами для ИИ-агентов. Они **не требуются** для сборки, тестирования, релиза или работы самого Flutter-приложения. Проект успешно собирается стандартными средствами Flutter CLI.

---


## 1. Рекомендованные MCP-серверы

1. **Dart and Flutter MCP Server (РЕКОМЕНДУЕТСЯ):**
   * Предоставляет доступ к DTD (Dart Tooling Daemon), анализу файлов через LSP (Language Server Protocol) и запуску тестов.
   * Позволяет выполнять точный рефакторинг и инспектировать виджеты.
2. **Context7 MCP (ОПЦИОНАЛЬНО):**
   * Используется для получения свежей документации по пакетам с pub.dev в реальном времени.
3. **Filesystem MCP (С ОГРАНИЧЕНИЯМИ):**
   * Предоставляет доступ на чтение/запись файлов. **Ограничение:** Разрешать доступ только к корневой директории проекта Kopim, без доступа к домашней директории пользователя (`/home/artem/`).
4. **GitHub MCP (ОПЦИОНАЛЬНО):**
   * Допускается только для чтения репозитория и создания PR. **Ограничение:** Не давать прав на удаление веток или изменение настроек безопасности репозитория.
5. **Firebase/Production Data MCP (ЗАПРЕЩЕНО):**
   * **Категорически запрещено** подключать MCP, имеющие прямой доступ к боевым (production) данным Firestore или Auth без предварительного safety-аудита и согласования с командой.

---

## 2. Примеры конфигурации для Codex CLI

Если в вашей системе установлен Codex CLI, вы можете добавить Dart MCP-сервер следующей командой:

```bash
codex mcp add dart -- dart mcp-server --force-roots-fallback
```

---

## 3. Примеры настройки для IDE и Инструментов

### Cursor (`.cursor/config.json` или в настройках UI)
Для интеграции Dart MCP-сервера в Cursor перейдите в `Settings -> Features -> MCP` и добавьте новый сервер:
* **Name:** `dart-mcp`
* **Type:** `command`
* **Command:** `dart mcp-server --force-roots-fallback`

### Claude Code / Gemini Code Assist (или аналогичные консольные агенты)
В файле конфигурации MCP серверов (обычно `mcp.json` или `claude_desktop_config.json`) добавьте блок:

```json
{
  "mcpServers": {
    "dart-mcp": {
      "command": "dart",
      "args": [
        "mcp-server",
        "--force-roots-fallback"
      ]
    },
    "context7": {
      "command": "npx",
      "args": [
        "-y",
        "@context7/mcp-server-pub-dev"
      ]
    }
  }
}
```
