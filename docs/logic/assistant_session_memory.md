# Память диалога ассистента (сессия)

## Назначение
Сохранять контекст текущего диалога ИИ-ассистента в пределах одной сессии,
чтобы последующие вопросы опирались на предыдущие ответы без сохранения в БД.

## Параметры
- Лимит истории: 10 последних сообщений (user/assistant).
- Источник: состояние сессии ассистента (Riverpod).
- Исключения: системные и недоставленные сообщения не учитываются.

## Пример использования
1) Пользователь: "Сколько я потратил в этом месяце?"
2) Ассистент: "100 ₽"
3) Пользователь: "Сколько я потратил, если не считать продукты?"
Ассистент видит историю и уточняет запрос, опираясь на прошлый ответ.

## Реализация
- Контекст диалога добавляется в текст запроса перед текущим вопросом.
- Контекст включает последние сообщения, формат: "Пользователь/Ассистент: ...".
