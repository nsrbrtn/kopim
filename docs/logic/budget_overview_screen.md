# Экран конкретного бюджета (Budget Overview)

## Что изменилось
- Добавлен отдельный экран конкретного бюджета с секциями: сводка, динамика расходов, категории с раскрытием транзакций, предстоящие платежи.
- Для бюджета по категориям breakdown теперь строится по explicit allocation-узлам, а подкатегории отображаются вложенно внутри родителя.
- Родительская категория в breakdown показывает только свой direct-лимит и direct-траты; лимиты и траты explicit-подкатегорий больше не дублируются в строке родителя.
- Переход на экран выполняется по тапу на карточку бюджета на главном экране и на экране списка бюджетов.

## Зачем
- Разделить короткий виджет бюджета и полноценный экран деталей.
- Дать быстрый обзор расходов и ближайших платежей с фильтром по периоду.
- Устранить двойной учет в сценарии, когда у родителя и подкатегорий заданы собственные лимиты.

## Как проверить
- Открыть главный экран и тапнуть по виджету бюджета.
- Переключить «Месяц/Неделя» и убедиться, что суммы и график меняются.
- Открыть экран бюджетов и тапнуть по карточке бюджета.
- Создать бюджет, где у родителя есть лимит, а у части подкатегорий есть собственные лимиты.
- Раскрыть родительскую категорию и убедиться, что внутри нее показаны подкатегории с отдельными лимитами.
- Проверить, что в строке родителя отображается direct-лимит: например, `10000 - 3000 = 7000`.
- Убедиться, что транзакции explicit-подкатегории отображаются только внутри нее, а не дублируются в транзакциях родителя.
- Проверить блок «Предстоящие платежи».
