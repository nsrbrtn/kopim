# Экран профиля и настройки учетной записи

## Что изменилось

- Основной экран профиля (`/profile`) перестроен под обзорный формат:
  - хедер с аватаром, приветствием и кнопкой-шестеренкой;
  - блок активности за неделю;
  - блок прогресса уровня;
  - блок «Данные и безопасность»;
  - кнопка «Выйти из профиля».
- Секция `Учетная запись` (имя, валюта, локаль, сохранение) перенесена с экрана профиля на экран `Генеральные настройки`, который открывается по шестеренке.
- Внутри секции `Учетная запись` добавлено destructive-действие `Удалить аккаунт` с подтверждением через кодовое слово и текущий пароль.

## Зачем

- Разделить обзор профиля и редактирование учетных данных.
- Привести структуру экрана профиля к новому макету.

## Как проверить

1. Открыть профиль из аватара на главном экране.
2. Убедиться, что на профиле отсутствует секция `Учетная запись`.
3. Нажать на шестеренку в правом верхнем углу:
   - открывается экран `Генеральные настройки`;
   - первой секцией отображается `Учетная запись` с полями, кнопкой сохранения и действием удаления аккаунта.
4. Нажать `Удалить аккаунт`:
   - открывается подтверждающий диалог;
   - destructive-кнопка активируется только после ввода кодового слова и текущего пароля.
5. Вернуться на профиль и проверить:
   - отображение активности, прогресса и секции данных/безопасности;
   - кнопку выхода из профиля.

## Breaking changes
