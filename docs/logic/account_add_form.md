# Форма добавления счета

## Что изменилось
- Поле начального баланса отображается для всех типов счета, включая `Кредитная карта`.
- При сохранении нового счета введенное значение начального баланса применяется к `balanceMinor` и `openingBalanceMinor`.

## Зачем
- Убрать расхождение UX между типами счетов.
- Дать возможность задать стартовый баланс кредитной карты при создании, а не фиксировать `0`.

## Как проверить
- Открыть экран добавления счета и выбрать тип `Кредитная карта`.
- Убедиться, что поле «Начальный баланс» доступно для ввода.
- Ввести валидный баланс, заполнить обязательные поля кредитной карты и сохранить.
- Проверить, что счет создается без ошибки и стартовый баланс отображается в данных счета.

## Breaking changes
- Нет.
