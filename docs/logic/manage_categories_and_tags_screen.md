# Экран «Категории и тэги»

## Что изменилось

- Заголовок экрана изменён на `Категории и тэги`.
- Вкладка категорий теперь показывает плоский список:
  - родительские категории;
  - подкатегории сразу под родителем (без раскрывающихся секций).
- Подкатегории визуально уменьшены относительно родительских категорий и имеют дополнительный отступ.
- Удалены действия по свайпу влево/вправо для категорий и подкатегорий.
- Редактирование категории и подкатегории открывается по тапу на элемент списка.
- В экран редактирования категории добавлена кнопка удаления (работает и для подкатегорий).

## Зачем

- Сделать управление категориями более предсказуемым: без скрытых уровней и без жестов, которые сложно обнаружить.
- Перенести критичное действие удаления в явный UI на экране редактирования.

## Как проверить

1. Открыть экран управления категориями из профиля.
2. Убедиться, что заголовок экрана: `Категории и тэги`.
3. На вкладке `Категории` проверить:
   - список не содержит раскрывающихся секций;
   - подкатегории отображаются сразу после родителя;
   - подкатегории визуально меньше.
4. Провести свайп по категории и подкатегории: свайп-действия отсутствуют.
5. Тапнуть по категории и подкатегории: открывается экран редактирования.
6. В режиме редактирования существующей категории/подкатегории проверить наличие кнопки `Удалить` и успешное удаление после подтверждения.

## Ограничения

- Создание подкатегории по-прежнему доступно только для корневых категорий.
