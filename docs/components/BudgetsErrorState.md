# BudgetsErrorState

## Назначение
Виджет состояния ошибки для экрана бюджетов. Отображается когда произошла ошибка при загрузке данных о бюджетах.

> **Примечание**: Это приватный виджет (`_BudgetsErrorState`), используемый внутри экрана бюджетов.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `message` | `String` | Текст сообщения об ошибке | *обязательный* |
| `strings` | `AppLocalizations` | Объект локализации для получения переведённых строк | *обязательный* |

## Особенности поведения

- **Центрированная компоновка**: содержимое размещается по центру экрана
- **Иконка предупреждения**: крупная иконка `warning_amber_rounded` красного цвета (`error`)
- **Два текстовых блока**:
  - Заголовок из локализации (общий заголовок ошибки)
  - Сообщение об ошибке (переданное через параметр `message`)
- **Адаптивные отступы**: использует систему spacing из `KopimLayout`
- **Адаптивный размер иконки**: размер иконки берётся из `iconSizes.lg`

## Визуальная структура

```
┌─────────────────┐
│                 │
│       ⚠️        │ ← иконка (LG, красная)
│                 │
│   Заголовок     │ ← titleMedium
│                 │
│   Сообщение     │ ← bodySmall
│   об ошибке     │
│                 │
└─────────────────┘
```

## Пример использования

```dart
// Используется внутри экрана бюджетов
if (errorMessage != null) {
  return _BudgetsErrorState(
    message: errorMessage,
    strings: AppLocalizations.of(context)!,
  );
}

// Пример с конкретной ошибкой
_BudgetsErrorState(
  message: 'Не удалось загрузить данные. Проверьте подключение к интернету.',
  strings: strings,
)
```
