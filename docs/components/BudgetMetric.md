# BudgetMetric

## Назначение
Вспомогательный виджет для отображения метрики бюджета в формате "лейбл + значение". Используется внутри `BudgetCard` для отображения таких данных как "Потрачено", "Лимит" и "Остаток/Превышено".

> **Примечание**: Это приватный виджет (`_BudgetMetric`), используемый только внутри `BudgetCard`. Документация приведена для справки.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `label` | `String` | Текст метки (например, "Потрачено", "Лимит") | *обязательный* |
| `value` | `String` | Значение метрики (обычно отформатированная сумма) | *обязательный* |
| `valueStyle` | `TextStyle?` | Кастомный стиль для значения (для выделения, например, красным) | `null` |

## Особенности поведения

- **Вертикальная компоновка**: метка сверху, значение снизу
- **Стиль метки**: использует `bodySmall` с цветом `onSurfaceVariant` (приглушённый)
- **Стиль значения**: по умолчанию `bodyMedium`, но может быть переопределён через `valueStyle`
- **Отступ**: между меткой и значением — половина стандартного отступа `spacing.between`

## Пример использования

```dart
_BudgetMetric(
  label: 'Потрачено',
  value: '22 500 ₽',
)

// С кастомным стилем (для превышения бюджета)
_BudgetMetric(
  label: 'Превышено',
  value: '2 500 ₽',
  valueStyle: TextStyle(
    color: Colors.red,
    fontWeight: FontWeight.w600,
  ),
)
```
