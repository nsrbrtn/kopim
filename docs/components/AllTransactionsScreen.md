# AllTransactionsScreen

## Назначение
Экран полного списка транзакций с фильтрами по дате, счёту и категории.

## Секции и поведение
- AppBar: заголовок `Все транзакции`.
- FAB: открывает создание новой транзакции прямо с экрана списка.
- Фильтры: дата, счёт, категория и быстрый сброс.
- Список: группировка по дням, отдельная карточка на каждую транзакцию или групповой платёж по кредиту.

## Что изменилось
- На экран добавлена явная кнопка создания транзакции.

## Зачем
- Пользователь может создать новую запись, не возвращаясь на главный экран или в другой раздел.

## Как проверить
1. Открыть экран `Все транзакции`.
2. Нажать на FAB с `+`.
3. Убедиться, что открылась форма создания транзакции на mobile, desktop и web.
