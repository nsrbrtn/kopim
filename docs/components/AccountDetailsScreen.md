# AccountDetailsScreen

## Назначение
Экран обзора счёта после перехода с карточки на главной. Показывает динамику баланса, итоги периода, топ категорий и список транзакций.

## Секции и поведение
- Переключатель периода: месяц / квартал / год. Влияет на график, итоги и топ категорий.
- График баланса: показывает изменение остатка по выбранному периоду.
- Итоги периода: доходы, расходы и итог за период.
- Топ категорий: до 5 крупнейших категорий расходов с процентом и прогресс‑баром.
- Транзакции: фильтр «все/расходы/доходы» и список по дням, каждая транзакция в отдельном контейнере.
- История транзакций: по умолчанию показывается только текущий месяц.
- Кнопка `Показать больше`: подгружает в список ещё один предыдущий месяц истории за каждое нажатие, пока пользователь не дойдёт до всех транзакций счёта.
- FAB: добавление транзакции по умолчанию в текущий счёт.
- Web/desktop responsive:
  - маршрут `/accounts/details` получает расширенный контейнер вместо глобального лимита `600px`;
  - внутренние горизонтальные отступы считаются от реально доступной ширины контейнера, а не от ширины окна;
  - summary-блок переключается в компактный вертикальный layout на узкой ширине, чтобы длинные суммы и подписи не выходили за границы.

## Что изменилось
- Список транзакций больше не ограничен переключателем периода экрана.
- Для списка введено отдельное окно видимой истории: стартует с текущего месяца и расширяется помесячно по кнопке.

## Зачем
- На длинной истории счёта экран остаётся компактным по умолчанию.
- При этом пользователь может дойти до любой старой операции без смены экрана и без потери текущего контекста счёта.

## Как проверить
1. Открыть счёт с историей длиннее одного месяца.
2. Убедиться, что сначала видны только транзакции текущего месяца.
3. Нажать `Показать больше` и проверить, что появились транзакции предыдущего месяца.
4. Повторять нажатие, пока не станет доступна вся история счёта.
5. Переключить период `Месяц / Квартал / Год` и убедиться, что график и сводки меняются, а история раскрывается отдельно.
