# AccountDetailsScreen

## Назначение
Экран обзора счёта после перехода с карточки на главной. Показывает динамику баланса, итоги периода, топ категорий и список транзакций.

## Секции и поведение
- Переключатель периода: месяц / квартал / год. Влияет на график, итоги, топ категорий и список транзакций.
- График баланса: показывает изменение остатка по выбранному периоду.
- Итоги периода: доходы, расходы и итог за период.
- Топ категорий: до 5 крупнейших категорий расходов с процентом и прогресс‑баром.
- Транзакции: фильтр «все/расходы/доходы» и список по дням, каждая транзакция в отдельном контейнере.
- FAB: добавление транзакции по умолчанию в текущий счёт.
