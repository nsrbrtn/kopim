# TotalMoneyChartWidget

## Назначение
Виджет для отображения динамики общего баланса («Денег всего») по времени. Визуализирует максимальный баланс за каждый месяц с использованием плавной кривой (Spline).

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `data` | `List<MonthlyBalanceData>` | Данные о балансе по месяцам (MoneyAmount: minor/scale) | *обязательный* |
| `currencySymbol` | `String` | Символ валюты | *обязательный* |
| `selectedMonth` | `DateTime` | Выбранный месяц для подсветки точки | *обязательный* |
| `onMonthSelected` | `ValueChanged<DateTime>` | Коллбэк при нажатии на точку графика | *обязательный* |
| `localeName` | `String` | Локаль для форматирования | *обязательный* |

## Особенности поведения

- **Визуализация**: использует `SplineSeries` для создания мягкой кривой изменения баланса.
- **Подсветка выбора**: выбранная точка выделяется анимированным свечением (glow effect).
- **Масштабирование**: автоматически рассчитывает диапазон оси Y так, чтобы график занимал оптимальное пространство с небольшими отступами сверху и снизу.
- **Крупный заголовок**: отображает название выбранного месяца и точную сумму баланса крупным шрифтом.
- **Зона нажатия**: имеет увеличенные невидимые зоны клика (`ScatterSeries`) вокруг каждой точки для удобства взаимодействия на мобильных устройствах.

## Пример использования

```dart
TotalMoneyChartWidget(
  data: balanceHistory,
  currencySymbol: '$',
  selectedMonth: DateTime(2025, 12),
  onMonthSelected: (date) => updateFilters(date),
  localeName: 'en_US',
)
```
