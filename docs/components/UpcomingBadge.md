# UpcomingBadge

## Назначение
Приватный вспомогательный виджет для отображения бейджа с количеством предстоящих событий (платежей или напоминаний). Используется внутри `_UpcomingSummaryBadges` на главном экране.

> **Примечание**: Это приватный виджет (`_UpcomingBadge`), используемый только внутри компонента сводки предстоящих платежей. Документация приведена для справки.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `count` | `int` | Количество событий для отображения на бейдже | *обязательный* |
| `icon` | `IconData` | Иконка события (например, `Icons.event_repeat` или `Icons.alarm`) | *обязательный* |
| `color` | `Color` | Цвет бейджа и иконки | *обязательный* |
| `label` | `String` | Текстовая метка (например, "Платежи", "Напоминания") | *обязательный* |
| `semanticsLabel` | `String` | Описание для специальных возможностей accessibility | *обязательный* |

## Особенности поведения

- **Адаптивный цвет текста**: автоматически выбирает чёрный или белый цвет для текста на бейдже в зависимости от яркости фонового цвета
- **Числовой бейдж**: использует встроенный `Badge.count` для отображения количества
- **Семантика**: корректно работает со screen readers благодаря `Semantics`
- **Компактная компоновка**: иконка с бейджем + текстовая метка в одну строку

## Пример использования

```dart
_UpcomingBadge(
  count: 5,
  icon: Icons.event_repeat,
  color: Colors.blue,
  label: 'Платежи',
  semanticsLabel: 'У вас 5 предстоящих платежей',
)

_UpcomingBadge(
  count: 2,
  icon: Icons.alarm,
  color: Colors.orange,
  label: 'Напоминания',
  semanticsLabel: 'У вас 2 активных напоминания',
)
```
