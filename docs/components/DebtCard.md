# DebtCard

Карточка для отображения записи долга: сумма, счет, дата платежа и описание.

## Когда использовать

- В списке долгов внутри раздела "Кредиты и долги".
- В компактных списках без детальной аналитики.

## Параметры

- `debt` — доменная сущность долга.
- `onTap` — обработчик нажатия (опционально).

## Пример

```dart
DebtCard(
  debt: debt,
  onTap: () => context.push('/credits/debts/edit', extra: debt),
)
```
