# SettingsActionButton

## Назначение
Круглая кнопка с иконкой шестерёнки для открытия настроек приложения. Используется на экране профиля/меню.

> **Примечание**: Это приватный виджет (`_SettingsActionButton`), используемый внутри экрана меню.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `tooltip` | `String` | Текст подсказки при долгом нажатии | *обязательный* |
| `onPressed` | `VoidCallback` | Коллбэк при нажатии на кнопку | *обязательный* |

## События и коллбэки

| Событие | Описание |
|---------|----------|
| `onPressed()` | Вызывается при нажатии на кнопку — обычно открывает экран настроек |

## Особенности поведения

- **Иконка**: фиксированная иконка `Icons.settings_outlined` размером 28px
- **Форма**: круглая кнопка с тенью
- **Цвета**: 
  - Фон — `primary` из темы
  - Иконка — `onPrimary` из темы
- **Тень**: мягкая тень с цветом кнопки (alpha 0.35), смещение (0, 6), размытие 16
- **Доступность**: корректно работает со screen readers благодаря `Semantics`
- **Tooltip**: отображается при долгом нажатии

## Пример использования

```dart
_SettingsActionButton(
  tooltip: 'Открыть настройки',
  onPressed: () {
    Navigator.push(
      context,
      MaterialPageRoute(
        builder: (_) => SettingsScreen(),
      ),
    );
  },
)
```
