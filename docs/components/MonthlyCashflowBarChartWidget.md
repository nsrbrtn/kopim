# MonthlyCashflowBarChartWidget

## Назначение
Виджет для визуализации денежных потоков (доходы vs расходы) по месяцам. Позволяет сравнивать финансовые показатели за разные периоды и видеть чистый остаток.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `data` | `List<MonthlyCashflowData>` | Список данных о денежных потоках по месяцам | *обязательный* |
| `currencySymbol` | `String` | Символ валюты для отображения | *обязательный* |
| `selectedMonth` | `DateTime` | Текущий выбранный месяц | *обязательный* |
| `onMonthSelected` | `ValueChanged<DateTime>` | Коллбэк при выборе месяца на графике | *обязательный* |
| `localeName` | `String` | Локаль для форматирования дат и чисел | *обязательный* |

## Особенности поведения

- **Группировка данных**: отображает три типа столбцов для каждого месяца: Доход, Расход и Остаток.
- **Горизонтальный скролл**: поддерживает переключение между окнами по 6 месяцев с помощью свайпа.
- **Интерактивность**: нажатие на любую область месяца (включая невидимые хит-боксы) вызывает выбор этого месяца.
- **Информационный диалог**: содержит кнопку помощи с описанием логики расчета (Income - Expense).
- **Адаптация**: настраивает высоту и масштаб в зависимости от максимальных значений в наборе данных.

## Пример использования

```dart
MonthlyCashflowBarChartWidget(
  data: cashflowData,
  currencySymbol: '₽',
  selectedMonth: DateTime.now(),
  onMonthSelected: (date) {
    print('Выбран месяц: $date');
  },
  localeName: 'ru_RU',
)
```
