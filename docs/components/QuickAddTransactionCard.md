# QuickAddTransactionCard

Карточка быстрого добавления транзакции на главном экране.

## Поведение

- Показывает только избранные категории.
- Если избранных категорий нет, вместо списка иконок отображает плейсхолдер:
  `Чтобы быстро создать операцию, добавьте категорию в избранное.`
- После успешного быстрого добавления показывает `SnackBar` c длительностью `1` секунда.

## Как проверить

1. Открыть главный экран с избранными категориями и убедиться, что в карточке видны только они.
2. Удалить флаг `избранное` у всех категорий и убедиться, что вместо списка отображается плейсхолдер.
3. Нажать на избранную категорию, сохранить транзакцию и проверить, что success-snackbar закрывается через одну секунду.
