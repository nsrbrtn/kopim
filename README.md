
#  финансовое приложение

![Kopim App](https://github.com/user-attachments/assets/18413348-1549-4a6c-a1d2-09439c298059)

**Копим** — это современное, масштабируемое и безопасное приложение для управления личными финансами. Проект разработан с использованием передовых технологий и лучших практик в экосистеме Flutter.

## ✨ Особенности

- **Чистая архитектура + DDD:** кодовая база разделена на изолированные модули по фичам (`feature-first`), что упрощает навигацию и поддержку.
- **Офлайн-режим:** приложение работает даже без интернета благодаря локальной базе данных Drift. Синхронизация с облаком происходит автоматически при появлении сети.
- **Современный стек:** использование Riverpod для управления состоянием, Freezed для иммутабельных моделей и Firebase для бэкенда.
- **Кроссплатформенность:** одна кодовая база для Android, iOS, Web и Desktop.

## 🛠️ Стек технологий

- **Фреймворк:** Flutter 3.38+ (с поддержкой Dot Shorthands и Impeller)
- **Управление состоянием:** Riverpod
- **Модели данных:** Freezed
- **Локальная база данных:** Drift (SQLite)
- **Бэкенд и синхронизация:** Firebase (Auth, Firestore, Performance)
- **Архитектура:** Clean Architecture + Domain-Driven Design (DDD)
